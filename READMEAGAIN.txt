How to set up E-health database on your local host

1.Open http://localhost/phpmyadmin/

2.Creat a new database name as "wp"

3.Import the "_wp_.sql" databse file into "wp" database

4.Add a database user account with the following info:

username: magic4
hostname: choose - localhost
password: magic4
Global privileges: Check all

Admin username and password should be: magic4 and magic4magic4
